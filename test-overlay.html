Click the gear button to toggle the controls overlay

Actions Controls

This would contain the ActionsControls component with buttons for:

  • Rerender
  • New Palette
  • Randomize
  • Show Guides toggle
  • Global controls (Randomness, Stroke Count, Color Bleed)

Ring Controls

This would contain the RingsControls component with:

  • Individual ring controls
  • Grammar input fields
  • Symbol-specific controls
  • Parameter sliders

Responsive Layout

This layout is responsive:

  • Mobile: 1 column (full width)
  • Tablet: 2 columns
  • Desktop: 3 columns

Try resizing your browser window to see the layout change!

Blur Effects: You should see translucency and blur effects on the overlay and control boxes.
